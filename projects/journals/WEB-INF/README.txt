To install this webapp, try using Tomcat /manager/html

1. Edit the path to your journal directory for @Context_journals.xml@
2. Use http://localhost:8080/manager/html
3. Fill out the Deploy form:
	Context Path (required): /journals
	XML Configuration file URL: file:///D:/source/journals/WEB-INF/Context_journals.xml
4. Hit Deploy button.
